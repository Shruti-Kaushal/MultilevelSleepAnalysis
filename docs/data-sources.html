2.1 MMASH

There are 22 folders in the MMASH dataset, and each contains 7 files with the information of the user.

Each user folder contains:

  1. Actigraph.csv
  2. Activity.csv
  3. questionnaire.csv
  4. RR.csv
  5. saliva.csv
  6. sleep.csv
  7. user_info.csv
